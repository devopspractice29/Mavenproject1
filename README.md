# Mavenproject1
example
Example Maven project generated using maven-archetype-quickstart


Release

To use it in your Maven build add:

  <repositories>
	<repository>
	    <id></id>
	    <url></url>
	</repository>
  </repositories>
and the dependency:

	<dependency>
		<groupId>com.github.xx</groupId>
		<artifactId>maven-simple</artifactId>
		<version>0.1</version>
	</dependency>
